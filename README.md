# xzyNotepad
An open-source versatile notepad designed specifically for writing and editing code in various programming languages.

## Used modules:
- customtkinter
- pygments
- tkinter
- CTkMessagebox
- CTkMenuBar
- CTkCodeBox
- os
- json
- requests
